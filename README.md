# 📈 NIFTY Stocks AI Dashboard

An interactive **Streamlit-based stock analysis dashboard** with data visualization and machine learning prediction using **Random Forest**.

https://niftystocksanalysis.streamlit.app/

---

## 🚀 Features

- 📊 **Market Overview (All Stocks)**
  - Yearly Growth (Sorted)
  - Top Gainers 🚀
  - Top Losers 📉
  - Correlation Heatmap 🔥

- 📈 **Company-Level Insights**
  - Price, High, Low metrics
  - Filter by company

- 🤖 **AI Prediction**
  - Random Forest model
  - Predict stock price based on:
    - Open
    - High
    - Low
    - Volume

- 📄 **Data Viewer**
  - Expandable table for raw data

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Visualization:** Plotly  
- **Backend:** Python  
- **ML Model:** Random Forest (Scikit-learn)  
- **Data Handling:** Pandas, NumPy

## 🌐 Deployment

This app can be deployed using **Streamlit Cloud**:

1. Push code to GitHub  
2. Go to https://share.streamlit.io  
3. Select your repository  
4. Deploy 🚀  

---

## 📊 Dataset

- Source: NSE (National Stock Exchange of India)
- Contains:
  - Open, High, Low, Price
  - Volume (lacs)
  - Yearly & Monthly % change
  - 52-week high/low

---

## ⚠️ Limitations

- This model uses **static data (not time-series)**
- Predictions are **not suitable for real trading**
- No real-time data integration
